# Overview
This repository is a personal collection of programming projects/assignments from my university. It's both an archive of my academic work and a portfolio of technical skills I've developed during my degree.
More stuff will be added and this README is subject to change.

I entered USATU (Applied Math) in 2022. A merger soon changed my faculty.
Teaching was terrible. By semester 5, I took academic leave. That year became my most productive: I filled gaps in math, coded a lot, and found lectures by C. Muratori & J. Blow (via algorithmic luck). They shaped me deeply.

When I returned, the program had changed – I had even more debt.
I managed to transfer and now I’m in year 2 of a 5‑year specialist degree in Infocommunication Technologies and Communication Systems.

Unfortunately this repo holds work only from the new major.
